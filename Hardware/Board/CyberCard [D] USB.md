### CyberCard ###

There are three board variants that can be made using the same PCB and varying
parts.


#### USB-to-Serial ####

This is a basic variant and provides USB serial port directly connected to UPS'
serial output.

##### Input (USB) #####

Voltage: 5 V
Current: 100 mA
